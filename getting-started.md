
# Introduction
Welcome to the Getting Started Guide. This manual helps end users install, configure, and use the application.

## Installation
Ensure you have **administrator privileges**. Download the installer from the customer portal and run it.

### System requirements
- Windows 10 or later / macOS 12 or later
- 4 GB RAM minimum
- Network access to `https://example.com`

## Configuration
Open *Settings* and configure preferences. Use `Ctrl + ,` to open Settings.

### Profiles
Create a profile for each environment. Profiles store endpoints, credentials, and feature toggles.

## Basic Usage
From the dashboard, select a project and click **Run**.

> **Tip:** Use the command palette (`Ctrl + K`) for quick actions.

### Troubleshooting
If something fails, check the logs under `~/AppData/YourApp/logs` or `~/Library/Logs/YourApp` on macOS.
